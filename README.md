# Introduction

作为 Lain 集群的白皮书，会对 Lain 的安装部署/使用/参与开发等方方面面给出充分的说明。

### LAIN’s An Infrastructure Network
### LAIN Ain’t Infrastructure Network

Lain 是一个基于 docker 的 PaaS 系统。

设计目标包括但不限于：

- 降低系统管理复杂度
- 简化服务的部署管理
- 优化基础服务的调配
- 提高资源的使用效率
- 统一开发测试生产三环境
- 持续交付工作流的良好支持
